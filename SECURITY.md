# Chính sách bảo mật

## Báo cáo lỗ hổng bảo mật

Vui lòng không tạo issue công khai đối với các lỗ hổng bảo mật.

Hãy báo cáo riêng tư thông qua GitHub Security Advisories:

1. Mở repository trên GitHub.
2. Truy cập **Security** > **Advisories**.
3. Chọn **Report a vulnerability**.

Vui lòng cung cấp các bước tái hiện lỗi, thành phần bị ảnh hưởng và mức độ tác động dự kiến.
