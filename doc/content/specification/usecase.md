## UC01 - Login

### Ten Use Case
Login

### Mo ta
User dang nhap vao he thong de su dung cac chuc nang quan ly va scan menu.

### Actor
User

### Tien dieu kien
- User da co tai khoan.
- User chua dang nhap.

### Hau dieu kien
- User dang nhap thanh cong va duoc chuyen vao trang chinh.

### Luong chinh
1. User nhap email va mat khau.
2. User bam nut Login.
3. He thong kiem tra thong tin dang nhap.
4. He thong tao phien dang nhap.
5. He thong chuyen User vao trang chinh.

### Luong thay the
- Neu User quen mat khau, User co the chon chuc nang reset password.

### Luong ngoai le
- Neu email hoac mat khau sai, he thong hien thi thong bao loi.
- Neu tai khoan bi khoa, he thong khong cho dang nhap.
